## 自定义输入框获取焦点

```
<template>
    <textarea v-focus class="edit-area" v-model="applyTitle" placeholder="请输入标题"></textarea>
</template>
<script>
    export default {
        // 自定义输入框获取焦点方法
        directives: {
            focus: {
                inserted: (el) => {
                    el.focus()
                }
            }
        }
    }
</script>
```
