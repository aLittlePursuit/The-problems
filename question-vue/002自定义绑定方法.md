## 自定义绑定方法
> 自定义输入框获取焦点方法 v-foucs

```
<template>
    <textarea v-focus class="edit-area" v-model="applyTitle" placeholder="请输入标题"></textarea>
</template>
<script>
    export default {
        // 自定义输入框获取焦点方法
        directives: {
            focus: {
                inserted: (el) => {
                    el.focus()
                }
            }
        }
    }
</script>
```
> 自定义绑定better-scroll方法 v-scroll

```
<template>
    <div class="content" v-scroll>
        <ul class="bound-scroll">
            <li v-for="item in 100">{{ item }}</li>
        </ul>
    </div>
</template>
<script>
    // 首先引入插件better-scroll
    import BSscroll from 'better-scroll'

    export default {
        // 自定义输入框获取焦点方法
        directives: {
            scroll: {
                inserted: (el) => {
                    const scroll = new BScroll(el, {
                        probeType: 3,
                        click: true
                    })
                    scroll.refresh()
                }
            }
        }
    }
</script>
```
